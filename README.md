# SteeringAnalysis

Python 3.13 analysis project using `uv`.

## Project Structure

```
SteeringAnalysis/
├── src/        # Source code
├── scripts/    # Run scripts and entrypoints
├── tests/      # Test suite
├── docs/       # Experiment descriptions
├── results/    # Experiment output and results
└── .ignore     # Files searchable by OpenCode (gitignored)
```

## Setup

Requires Python ≥3.13 and [uv](https://docs.astral.sh/uv/).

```bash
uv sync
```
